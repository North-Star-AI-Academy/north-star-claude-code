# Claude Code Workshop

> **North Star AI Academy** — Claude Code: The Full Course 2026

Een hands-on workshop repo voor marketeers die Claude Code leren gebruiken. Kloon deze repo tijdens de live sessie en werk door 8 levels heen — van installatie tot een eigen herhaalbaar proces.

## Wat zit erin?

| Map | Wat | Voor wie |
|---|---|---|
| `exercises/` | 8 levels met opdrachten | Workshop deelnemers |
| `.claude/skills/` | 15+ marketing skills | Iedereen die Claude Code gebruikt |
| `templates/` | CLAUDE.md per marketing rol | Kopiëren naar je eigen project |
| `mcp-configs/` | MCP server voorbeelden | Setup inspiratie |
| `cheatsheet/` | Quick reference card | Dagelijks gebruik |

## Quick Start

```bash
# 1. Kloon de repo
git clone https://github.com/NorthStarAIAcademy/claude-code-workshop.git
cd claude-code-workshop

# 2. Start Claude Code
claude

# 3. Begin de eerste oefening
/start-exercise 1
```

## De 8 Levels

| Level | Onderwerp | Tijd |
|---|---|---|
| 1 | Hello Claude — installatie en eerste commando's | 10 min |
| 2 | Project Memory — CLAUDE.md en regels | 15 min |
| 3 | Marketing Skills — skills activeren en gebruiken | 20 min |
| 4 | MCP Tools — externe tools verbinden | 15 min |
| 5 | Context Management — je context window als budget | 15 min |
| 6 | Git Workflows — branches, worktrees, automation | 15 min |
| 7 | Build Your Process — een herhaalbaar werkproces | 15 min |
| 8 | The Minimum Stack — wat je écht nodig hebt | 10 min |

## Skills gebruiken in je eigen projecten

Kopieer de hele `.claude/skills/` map naar je eigen project:

```bash
cp -r .claude/skills/ /pad/naar/je-project/.claude/skills/
```

Of kopieer individuele skills:

```bash
cp .claude/skills/copywriting.md /pad/naar/je-project/.claude/skills/
```

## CLAUDE.md templates

In `templates/` vind je kant-en-klare CLAUDE.md bestanden voor:

- **Marketing Manager** — team aansturen, campagnes plannen
- **Content Creator** — blogs, social, video scripts
- **Growth Marketer** — outbound, funnels, experiments
- **Agency** — multi-client, brand guidelines, deliverables
- **E-commerce** — productpagina's, ads, email flows

Kopieer er een naar de root van je project en pas aan.
