Controleer de voortgang van de student in de Claude Code workshop.

1. Kijk welke exercises al voltooid zijn (check of er output/resultaat bestanden zijn aangemaakt)
2. Bepaal op welk level de student zich bevindt
3. Toon een compact overzicht:
   - Voltooide levels (met vinkje)
   - Huidig level
   - Nog te doen
4. Geef een aanmoedigende opmerking

Check specifiek:
- Level 1: Is Claude Code werkend? Kan de student commando's uitvoeren?
- Level 2: Is er een CLAUDE.md aangemaakt of aangepast?
- Level 3: Zijn er skills gebruikt? Check git log voor skill-gerelateerde wijzigingen
- Level 4: Is er een .mcp.json aanwezig?
- Level 5: Begrijpt de student context management? (check of /compact is gebruikt)
- Level 6: Zijn er branches aangemaakt? Git worktrees?
- Level 7: Is er een herhaalbaar proces beschreven?
- Level 8: Is er een minimum stack document?
