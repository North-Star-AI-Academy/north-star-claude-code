# Level 1: Hello Claude — Get Claude Code Running

## Doel
Claude Code installeren, begrijpen hoe het werkt, en je eerste commando's uitvoeren.

## Opdrachten

### 1.1 — Eerste interactie
Open deze repo in je terminal en start Claude Code:
```bash
claude
```
Vraag Claude: "Wat zit er in deze repo?"

Claude leest automatisch het `CLAUDE.md` bestand en begrijpt de projectstructuur.

### 1.2 — Slash commands ontdekken
Probeer deze commando's:
- `/help` — Bekijk alle beschikbare commando's
- `/status` — Bekijk de status van je sessie
- `/compact` — Comprimeer je context (handig voor lange sessies)

### 1.3 — Bestanden lezen en bewerken
Vraag Claude:
- "Lees het README.md bestand en geef een samenvatting"
- "Maak een nieuw bestand `mijn-notities.md` met 3 dingen die ik vandaag wil leren"

### 1.4 — Permission modes begrijpen
Claude Code vraagt toestemming voor bepaalde acties. Je ziet dit in de terminal:
- **Lezen**: Altijd toegestaan
- **Schrijven**: Vraagt toestemming (tenzij in allowlist)
- **Bash commando's**: Vraagt toestemming

Tip: Druk op `y` om toe te staan, `n` om te weigeren.
