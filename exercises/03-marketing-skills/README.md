# Level 3: Marketing Skills — Je AI Marketing Toolkit

## Doel
Leer hoe Claude Code skills werken en gebruik ze voor echte marketing taken.

## Theorie

Skills zijn instructiebestanden in `.claude/skills/` die Claude automatisch activeert wanneer je een relevante vraag stelt. Deze repo bevat 12+ marketing skills:

| Skill | Wanneer |
|---|---|
| `copywriting` | Website copy schrijven |
| `cold-email` | Outbound email campagnes |
| `social-content` | LinkedIn, Twitter, Instagram posts |
| `email-sequence` | Drip campaigns en nurture flows |
| `seo-audit` | SEO problemen diagnosticeren |
| `ad-creative` | Google/Meta/LinkedIn ad copy |
| `content-strategy` | Content planning en topic clusters |
| `page-cro` | Conversie optimalisatie |
| `landing-page` | Landingspagina's bouwen |
| `competitor-analysis` | Concurrentieanalyse |
| `schema-markup` | Structured data voor SEO |
| `marketing-ideas` | Groeistrategieën en tactieken |
| `copy-editing` | Bestaande copy verbeteren |
| `ab-testing` | Experimenten opzetten |

## Opdrachten

### 3.1 — Copywriting skill
```
Schrijf homepage copy voor een fictief bedrijf dat AI-gestuurde email
marketing software verkoopt aan e-commerce bedrijven. Prijs: €99/maand.
```

Let op hoe Claude automatisch de copywriting skill activeert en de juiste structuur volgt.

### 3.2 — Cold email skill
```
Schrijf een cold email sequence (3 emails) voor een B2B SaaS bedrijf
dat project management software verkoopt aan marketing agencies.
De trigger: het bureau heeft net een nieuwe CMO aangenomen.
```

### 3.3 — Social content skill
```
Maak een LinkedIn post over de impact van AI op het werk van marketeers.
Tone: thought leader, niet salesy. Gebaseerd op eigen ervaring.
```

### 3.4 — Combineer skills
```
Ik lanceer volgende maand een nieuwe online cursus over email marketing
voor freelancers. Prijs: €197. Geef me:
1. Een landingspagina structuur met copy
2. Een launch email sequence (5 emails)
3. 3 LinkedIn posts om de launch aan te kondigen
4. Google Ads headlines en descriptions
```

Merk op hoe Claude meerdere skills combineert in één antwoord.

### 3.5 — Eigen skill maken
```
Maak een nieuwe skill in .claude/skills/ voor "webinar-promotie" —
een skill die helpt bij het promoten van live webinars. Denk aan:
email uitnodigingen, social posts, reminder sequences, en registratiepagina copy.
```

## Checklist

- [ ] Je begrijpt hoe skills automatisch activeren
- [ ] Je hebt skills gecombineerd voor een complexe taak
- [ ] Je hebt een eigen skill aangemaakt
