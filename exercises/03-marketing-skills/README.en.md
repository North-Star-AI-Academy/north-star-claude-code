# Level 3: Marketing Skills — Your AI Marketing Toolkit

## Goal
Learn how Claude Code skills work and use them for real marketing tasks.

## Theory

Skills are instruction files in `.claude/skills/` that Claude automatically activates when you ask a relevant question. This repo includes 12+ marketing skills:

| Skill | When to use |
|---|---|
| `copywriting` | Writing website copy |
| `cold-email` | Outbound email campaigns |
| `social-content` | LinkedIn, Twitter, Instagram posts |
| `email-sequence` | Drip campaigns and nurture flows |
| `seo-audit` | Diagnosing SEO issues |
| `ad-creative` | Google/Meta/LinkedIn ad copy |
| `content-strategy` | Content planning and topic clusters |
| `page-cro` | Conversion rate optimization |
| `landing-page` | Building landing pages |
| `competitor-analysis` | Competitive analysis |
| `schema-markup` | Structured data for SEO |
| `marketing-ideas` | Growth strategies and tactics |
| `copy-editing` | Improving existing copy |
| `ab-testing` | Setting up experiments |

## Exercises

### 3.1 — Copywriting skill
```
Write homepage copy for a fictional company that sells AI-powered email
marketing software to e-commerce businesses. Price: €99/month.
```

Notice how Claude automatically activates the copywriting skill and follows the right structure.

### 3.2 — Cold email skill
```
Write a cold email sequence (3 emails) for a B2B SaaS company
that sells project management software to marketing agencies.
The trigger: the agency just hired a new CMO.
```

### 3.3 — Social content skill
```
Create a LinkedIn post about the impact of AI on the day-to-day work
of marketers. Tone: thought leader, not salesy. Based on personal experience.
```

### 3.4 — Combine skills
```
I'm launching a new online course next month about email marketing
for freelancers. Price: €197. Give me:
1. A landing page structure with copy
2. A launch email sequence (5 emails)
3. 3 LinkedIn posts to announce the launch
4. Google Ads headlines and descriptions
```

Notice how Claude combines multiple skills into a single response.

### 3.5 — Create your own skill
```
Create a new skill in .claude/skills/ for "webinar-promotion" —
a skill that helps promote live webinars. Think about:
email invitations, social posts, reminder sequences, and registration page copy.
```
