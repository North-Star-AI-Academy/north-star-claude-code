# Level 8: The Minimum Stack — Wat je écht nodig hebt

## Doel
Bepaal je persoonlijke minimum stack en neem een kant-en-klare setup mee naar je eigen projecten.

## The Minimum Stack voor Marketeers (2026)

| Component | Wat | Waarom |
|---|---|---|
| **Claude Code** | Je AI partner | Begrijpt context, gebruikt tools, werkt met bestanden |
| **CLAUDE.md** | Project geheugen | Claude kent je project zonder 33K token waste |
| **3-5 Skills** | Marketing toolkit | Consistente, hoge kwaliteit output voor vaste taken |
| **1-2 MCP servers** | Externe connecties | Web research + je belangrijkste tool (CRM, GitHub, etc.) |
| **Git** | Versiebeheer | Alles traceerbaar, niets kwijt |
| **Product Context** | Brand fundament | Alle skills kennen je merk, doelgroep, en tone |

## Opdrachten

### 8.1 — Je minimum stack samenstellen
```
Help me mijn persoonlijke minimum Claude Code stack samenstellen.

Mijn rol: [jouw rol]
Mijn tools: [tools die je dagelijks gebruikt]
Mijn taken: [3-5 taken die je het meest doet]

Geef me:
1. Welke skills ik moet kopiëren uit deze workshop
2. Welke MCP servers ik moet instellen
3. Wat er in mijn CLAUDE.md moet staan
4. Welke custom commands ik moet maken
```

### 8.2 — Setup kopiëren naar je eigen project
```
Kopieer mijn gekozen setup (skills, CLAUDE.md template, mcp config)
naar een nieuwe map "mijn-setup/" die ik straks naar mijn eigen
project kan kopiëren.
```

### 8.3 — Toekomstblik
```
Wat zijn de belangrijkste ontwikkelingen die eraan komen in Claude Code?
Denk aan:
- Nieuwe model capabilities
- Background agents
- Betere tool integraties
- Team features

En hoe kan ik me daar als marketeer op voorbereiden?
```

### 8.4 — Actieplan
```
Maak een 30-dagen actieplan voor mij om Claude Code te integreren
in mijn dagelijkse werk. Week per week, met concrete stappen.

Week 1: Setup en eerste taken
Week 2: Skills en workflows
Week 3: Automation en efficiency
Week 4: Optimalisatie en uitbreiding
```

## Wat je meeneemt

Na deze workshop heb je:
- [x] Claude Code draaiend en geconfigureerd
- [x] Een CLAUDE.md voor je project
- [x] Marketing skills die je direct kunt gebruiken
- [x] MCP tools voor research en integraties
- [x] Een herhaalbaar werkproces
- [x] Een 30-dagen actieplan

## Volgende stappen

1. **Kopieer je setup** naar je echte project
2. **Gebruik Claude Code dagelijks** — consistentie is key
3. **Bouw custom skills** voor je specifieke taken
4. **Deel met je team** — kopieer de `.claude/` map naar gedeelde repos
5. **Blijf leren** — Claude Code evolueert snel
