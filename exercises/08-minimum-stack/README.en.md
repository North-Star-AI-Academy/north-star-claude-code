# Level 8: The Minimum Stack — What you actually need

## Goal
Define your personal minimum stack and take a ready-to-go setup with you to your own projects.

## The Minimum Stack for Marketers (2026)

| Component | What | Why |
|---|---|---|
| **Claude Code** | Your AI partner | Understands context, uses tools, works with files |
| **CLAUDE.md** | Project memory | Claude knows your project without 33K token waste |
| **3-5 Skills** | Marketing toolkit | Consistent, high-quality output for recurring tasks |
| **1-2 MCP servers** | External connections | Web research + your most important tool (CRM, GitHub, etc.) |
| **Git** | Version control | Everything traceable, nothing lost |
| **Product Context** | Brand foundation | All skills know your brand, audience, and tone |
