# Level 7: Build Your Process — Een herhaalbaar werkproces

## Doel
Een persoonlijk, herhaalbaar proces opzetten dat je dagelijks kunt gebruiken met Claude Code.

## Het Ideale Proces

```
1. Start een nieuwe sessie (schone context)
2. Claude leest je CLAUDE.md (kent je project)
3. Je stelt een duidelijke vraag of taak
4. Claude gebruikt skills, tools, en bestanden
5. Je reviewt het resultaat
6. Commit en ga verder
```

## Opdrachten

### 7.1 — Je dagelijkse workflow ontwerpen
```
Help me een dagelijkse marketing workflow ontwerpen met Claude Code.
Ik ben een [jouw rol] bij [type bedrijf]. Mijn belangrijkste taken zijn:
- [taak 1]
- [taak 2]
- [taak 3]

Ontwerp een workflow die ik elke dag kan volgen met Claude Code,
inclusief welke skills ik gebruik en wanneer.
```

### 7.2 — Custom slash command maken
Maak een eigen slash command voor een taak die je vaak doet:

```
Maak een slash command .claude/commands/weekly-content.md die:
1. Mij vraagt naar het thema van de week
2. Een LinkedIn post genereert
3. Een Twitter thread maakt
4. Een korte blog outline schrijft
5. Social media captions voor Instagram maakt
Alles in mijn brand voice en consistent met mijn product-context.
```

### 7.3 — Superpowers ontdekken
Claude Code heeft unieke mogelijkheden die andere AI tools niet hebben:

```
Laat me de 5 grootste superpowers van Claude Code zien met een
praktisch marketing voorbeeld per superpower:
1. Multi-file editing (meerdere bestanden tegelijk aanpassen)
2. Sub-agents (parallelle taken)
3. Tool use (externe integraties)
4. Git integration (versiebeheer)
5. Persistent memory (CLAUDE.md + skills)
```

### 7.4 — Automation met hooks (bonus)
Hooks zijn scripts die automatisch draaien bij bepaalde events:

```
Leg uit hoe Claude Code hooks werken en geef 3 voorbeelden
die nuttig zijn voor marketeers:
- Een hook die spelling checkt voor elke commit
- Een hook die automatisch een changelog bijhoudt
- Een hook die brand guidelines checkt in nieuwe content
```
