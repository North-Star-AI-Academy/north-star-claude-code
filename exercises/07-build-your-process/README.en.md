# Level 7: Build Your Process — A repeatable workflow

## Goal
Set up a personal, repeatable process that you can use daily with Claude Code.

## The Ideal Process

```
1. Start a new session (clean context)
2. Claude reads your CLAUDE.md (knows your project)
3. You ask a clear question or assign a task
4. Claude uses skills, tools, and files
5. You review the result
6. Commit and move on
```

## Exercises

### 7.1 — Design your daily workflow
```
Help me design a daily marketing workflow with Claude Code.
I'm a [your role] at [type of company]. My main tasks are:
- [task 1]
- [task 2]
- [task 3]

Design a workflow I can follow every day with Claude Code,
including which skills I use and when.
```

### 7.2 — Create a custom slash command
Create your own slash command for a task you do frequently:

```
Create a slash command .claude/commands/weekly-content.md that:
1. Asks me for the theme of the week
2. Generates a LinkedIn post
3. Creates a Twitter thread
4. Writes a short blog outline
5. Creates social media captions for Instagram
All in my brand voice and consistent with my product context.
```

### 7.3 — Discover superpowers
Claude Code has unique capabilities that other AI tools don't have:

```
Show me the 5 biggest superpowers of Claude Code with a
practical marketing example for each:
1. Multi-file editing (modify multiple files at once)
2. Sub-agents (parallel tasks)
3. Tool use (external integrations)
4. Git integration (version control)
5. Persistent memory (CLAUDE.md + skills)
```

### 7.4 — Automation with hooks (bonus)
Hooks are scripts that run automatically on certain events:

```
Explain how Claude Code hooks work and give 3 examples
that are useful for marketers:
- A hook that spell-checks before every commit
- A hook that automatically maintains a changelog
- A hook that checks brand guidelines in new content
```
